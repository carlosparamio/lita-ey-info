# lita-ey-info

A handler for [Lita](http://lita.io/) to display EngineYard info.

## Installation

Add lita-ey-info to your Lita instance's Gemfile:

``` ruby
gem "lita-ey-info"
```

## Usage

```
ey envs
ey <app_name> <env_name> servers
ey <app_name> <env_name> logs
ey <app_name> <env_name> status
```
